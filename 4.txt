Кейс 4: Анализ оттока клиентов

Цель: Проанализировать клиентскую базу, рассчитать ключевые показатели удержания, выявить причины и динамику оттока.

Выполненные задачи:

Создана автоматизированная таблица клиентов с расчетом:
Статуса клиента (Новый/Активный/Ушедший)
Времени жизни клиента (в днях)
Сегментации по сумме покупок (Малый/Средний/Крупный)
Сегментации по активности (Недавние/Ядро/Ветераны)
Рассчитаны ключевые метрики:

Общее количество клиентов: 20
Уровень оттока: 50% (высокий показатель)
Cреднее время жизни клиента: 308 дней
Средняя сумма покупки: 63 705 ₽

Построен интерактивный дашборд, включающий:

Блок ключевых показателей с цветовой индикацией
Детальную сегментацию клиентов с долями в процентах
Динамику оттока по месяцам 2025 года
Три типа визуализации данных
Настроена автоматическая система: все показатели обновляются автоматически при изменении исходных данных.

Использованные навыки Microsoft Excel:

Логические и условные функции:
Вложенные функции ЕСЛИ для определения статусов
Функции И/ИЛИ для комплексных условий
СЧЁТЕСЛИ для подсчета клиентов по категориям

Работа с датами и временем:

Функция РАЗНДАТ для расчета времени жизни клиента
Группировка дат по месяцам в сводных таблицах
Функции МЕСЯЦ и ГОД для анализа динамики

Статистические расчеты:
Функция СУММПРОИЗВ для комплексных подсчетов
Расчет средних значений (СРЗНАЧ)
Работа с процентами и долями

Визуализация данных:

Круговая диаграмма для распределения клиентов по сумме покупок
Гистограмма для анализа сегментов по активности
Точечная диаграмма с линией тренда для отображения динамики оттока

Условное форматирование для индикации показателей

Проектирование интерфейса:

Создание структурированного дашборда
Компоновка блоков для удобства восприятия
Форматирование таблиц и диаграмм в едином стиле

Ключевые выводы из анализа:

Выявлена критическая ситуация: каждый второй клиент уходит (отток 50%)
Определены основные сегменты: большая часть клиентов относится к категории "Средний" по сумме покупок
Проанализирована динамика: отток распределен неравномерно по месяцам
Даны рекомендации: необходима программа лояльности и анализ причин оттока в проблемные месяцы
